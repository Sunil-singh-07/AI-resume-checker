\# AI Resume Checker



An AI-powered resume analyzer built with Flask and Google Gemini AI that gives instant feedback on your resume.



\## What it does

\- Upload your resume in PDF format

\- AI analyzes it instantly

\- Get detailed feedback on what to improve

\- Helps you make your resume job ready



\## Tech Stack

\- Python

\- Flask

\- Groq AI API

\- HTML/CSS



\## How to Run Locally

1\. Clone the repo

2\. Install dependencies - pip install -r requirements.txt

3\. Create a .env file and add your Groq API key - GROQ\_API\_KEY=your\_key\_here

4\. Run the app - python app.py

5\. Open https://ai-resume-checker-1anl.onrender.com/ in your browser
